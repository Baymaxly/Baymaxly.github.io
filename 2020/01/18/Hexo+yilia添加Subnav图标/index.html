Hexo + yilia添加Subnav图标


阅读量: 2020-01-18

yilia上面的Subnav社交地址还是很赞的，奈何偏偏没有CSDN的图标，搞得我也是头大呀

下面就来折腾一下如何添加CSDN的图标吧！

1、下载需要的图标

这里提供一个网站：https://www.easyicon.net/

大家可以从这里面直接搜索需要的图标，直接右键保存为jpg或者.png格式的就行

2、拷贝原有的iconfont.svg文件

进入到themes/yilia/source-src/css/fonts目录下

$ cd themes/yilia/source-src/css/fonts

将里面的iconfont.svg文件拷贝一份，这里我直接复制到博客根目录下

$ cp iconfont.svg /c/myblog

3、制作图片文字

这里推荐百度字体平台的字体编辑器：

http://fontstore.baidu.com/static/editor/index.html

先点左上角的新建，创建一个新项目；之后点上方的导入 –> 导入svg，将刚刚拷贝的iconfont.svg文件导入进去

完成之后如图：

之后继续选择导入 –> 导入图片，然后点击选择图片，找到 第一步保存的图片 ，添加进去之后视情况勾选反转（CSDN的需要勾选），然后确定就行。

之后点击导入图片的右上角的小画笔，进入编辑区

对图文进行全选，调整大小和位置，在右侧可以实时预览；调整完之后点击保存(S)；退出

选中图文，点击上方的字形信息，记住unicode编码，按照自己喜好进行命名

对空图文进行删除，然后点击上方的保存项目，之后点击ZIP进行下载

解压下载的文件夹，只保留名为fonteditor后缀为.eot,.svg,.ttf,.woff的四个文件，其他可以删除；

将剩下的文件名全部改为iconfont，后缀不变

4、修改相关配置文件

编辑themes/yilia/source-src/css/fonts路径下的fonts.scss文件：

在最后面添加.icon-上一步的重命名:before { content: "上一步的Unicode编码"; }

比如说我的是.icon-csdn:before { content: "\e878"; }

编辑同一路径下的social.scss文件，进行颜色的添加就行,比如说我的是

5、图文文件替换

将第三步得到的四个文件替换themes/yilia/source-src/css/fonts目录下的四个文件

6、打包

回到themes/yilia目录下$ cd ../../../

运行$ cnpm run dev用webpack打包，把文件编译到source文件里

如果报关于node sass的ERROR

先卸载再安装,之后再次打包：

打包成功后不会自动终止，此时按Ctrl + C进行强制终止：

然后执行进行发布：$ npm run dist，终止方法如上

然后就OKKKKKK啦，回到根目录：

$ hexo clean & hexo g & hexo s
